# ExorcistGame2046

Играбельный браузерный 3D-прототип кооперативного immersive sim про экзорцистов Санька и Прасковью в разделённом Ростове 2046 года.

## Что уже работает

- настоящий WebGL/Canvas 3D-срез от первого лица;
- разные квартиры героев: аскетичная квартира Санька и квартира-склад Прасковьи;
- инвентарь, лутинг, артефакты, телефон, штатив, селфи и внутренняя соцсеть;
- квест «Кровавая графиня» с читаемым ритуалом;
- процедурный центр города, патрули и уличные события;
- двухпользовательский P2P-кооператив через WebRTC;
- общий подъезд: Санёк живёт на втором этаже, Прасковья — на третьем;
- общий ФБК — Фонд по борьбе с крипосущностями;
- общая касса, канал «ФБК 2046», спонсоры и фанатская поддержка.

## Как запустить исправленную версию

### Windows

1. Скачайте репозиторий через **Code → Download ZIP** и распакуйте его полностью.
2. Запустите `START_GAME_WINDOWS.bat` или `serve.bat`.
3. Оставьте окно локального сервера открытым.
4. Браузер откроет `http://localhost:8080/play.html` либо следующий свободный порт.

Новый Windows-запуск не требует установленного Python: сначала используется встроенный PowerShell-сервер. Если он недоступен, батник пробует Python.

Для одиночной проверки без сервера можно открыть исправленный автономный HTML из релизного ZIP, приложенного в чате. Для WebRTC-кооператива используйте локальный HTTP-сервер.

### Через Node.js

```bash
npm start
```

Затем откройте:

```text
http://localhost:8080/play.html
```

### Linux / macOS

```bash
./serve.sh
```

## Исправление чёрного экрана — версия 0.1.1

В первой сборке отсутствовали базовые функции `clearWorld`, `buildCity` и `buildMission`, хотя меню и интерфейс продолжали отображаться. Поэтому после выбора героя игра падала, оставляя чёрный фон и HUD.

`play.html` загружает основной `index.html`, автоматически восстанавливает недостающие функции до выполнения игры и исправляет точки появления героев, чтобы камера не смотрела прямо в стену.

## Как подключить второго игрока

1. Оба игрока открывают `play.html` через localhost или HTTPS.
2. Выбирают разных героев: Санька и Прасковью.
3. Нажимают `K`.
4. Игрок 1 выбирает **HOST** и создаёт приглашение.
5. Игрок 2 выбирает **JOIN**, вставляет приглашение и создаёт ответ.
6. Игрок 1 вставляет ответ и применяет его.
7. После соединения оба увидят соседа и общую экономику ФБК.

## Управление

- `WASD` — движение;
- мышь — обзор;
- `E` — взаимодействие;
- `M` / `P` — телефон и 2ГИС;
- `I` — инвентарь;
- `1` — молитвенник;
- `2` — артефакт;
- `3` — ритуальный раствор;
- `4` / `L` — фонарик;
- `V` — селфи;
- `B` — стрим;
- `K` — кооператив, касса ФБК и канал;
- `Shift` — бег;
- `C` — присесть.

## Ветки разработки

- `main` — демонстрационная сборка;
- `integration/playable` — общая интеграция;
- `feature/coop-netcode` — сетевой слой;
- `feature/exorcism-combat` — бой и экзорцизм;
- `feature/world-sim` — город и события;
- `feature/interiors-inventory` — квартиры, лут и инвентарь;
- `feature/ui-phone-social` — UI, телефон и канал;
- `qa/playable-build` — проверки запуска.
